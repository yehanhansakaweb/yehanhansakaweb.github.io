Understanding Number Systems

A Number System is a structured way to represent numbers using digits or symbols. We commonly use systems like Binary, Decimal, Octal, and Hexadecimal, especially in computing and electronics.

Types of Number Systems

  • Binary (Base 2): Digits 0 and 1 only — the language of computers.
  • Decimal (Base 10): Digits 0–9 — the number system we use every day.
  • Octal (Base 8): Digits 0–7 — used in digital systems as a shortcut for binary.
  • Hexadecimal (Base 16): Digits 0–9 and A–F — compact binary representation.

Why We Use Different Systems

Computers process everything in binary. Hex and octal make binary shorter and easier to read. Decimal is used for human interaction. Understanding these systems helps bridge human-computer communication.

Conversions Between Systems

Binary ➝ Decimal

Multiply each binary digit by 2ⁿ from right to left, then add.

1011 → (1×8) + (0×4) + (1×2) + (1×1) = 11

Decimal ➝ Binary

Divide by 2 repeatedly, record remainders, then reverse them.

11 → 1011

Decimal ➝ Octal

Divide by 8, record remainders, and reverse them.

83 → 123

Octal ➝ Decimal

Multiply each digit by 8ⁿ from right to left, then add.

123 → 83

Decimal ➝ Hexadecimal

Divide by 16, record remainders, then convert to letters A–F.

254 → FE

Hexadecimal ➝ Decimal

Convert A–F to 10–15, multiply by 16ⁿ, then add.

FE → 254

Binary ➝ Hexadecimal

Group into 4 bits and convert each group to a hex digit.

11011111 → DF

Hexadecimal ➝ Binary

Convert each hex digit to 4-bit binary.

DF → 11011111

Binary ➝ Octal

Group into 3 bits and convert to octal digits.

110011 → 63

Octal ➝ Binary

Convert each digit to a 3-bit binary group.

63 → 110011

Conclusion

Number systems are foundational in computing. With the ability to convert between them, we better understand how computers process, store, and represent data efficiently.
